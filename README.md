# Aashish Web Designer - Bespoke Luxury Portfolio

A revolutionary, AI-driven portfolio experience for a next-gen freelance developer.

## 🚀 Vision
This isn't just a portfolio; it's a demonstration of what happens when AI meets elite engineering. The website acts as an intelligent assistant, helping businesses visualize their future digital presence in real-time.

## ✨ Key Features
- **AI Website Architect**: Generates live website blueprints based on business parameters using Gemini AI.
- **Interactive 3D Universe**: A Three.js powered hero section representing the digital landscape.
- **Industry Simulator**: Instant theme switching to showcase versatility across sectors (Restaurant, Startup, Gym, etc.).
- **Speed Revenue Simulator**: Interactive tool demonstrating the financial impact of performance.
- **Smart AI Contact**: A guided conversational interface that builds project briefs automatically.
- **Developer Mode**: Hidden console (Ctrl + `) for tech-savvy visitors.

## 🛠 Tech Stack
- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS (v4)
- **Animations**: Framer Motion (motion/react)
- **3D Rendering**: Three.js, @react-three/fiber, @react-three/drei
- **AI**: Google Gemini API (@google/genai)
- **Icons**: Lucide React

## 📦 Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables:
   - `GEMINI_API_KEY`: Your Google AI Studio API Key
4. Start development server: `npm run dev`

## 🚀 Deployment
This project is optimized for deployment on **Vercel** or **Netlify**.
- Build command: `npm run build`
- Output directory: `dist`

## 🎨 Design Language
- **Aesthetic**: Minimalist Luxury / Editorial
- **Typography**: Playfair Display (Headings), Inter (Body)
- **Palette**: Deep Black, Ivory White, Luxury Gold, Soft Gray
